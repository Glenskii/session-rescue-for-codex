# Product

## Register

product

## Users

Codex Desktop and Codex CLI users who accidentally archive a valuable task, need to audit local task state, or manage a large history by project. They are often responding to a stressful disappearance and need evidence before mutation.

## Product Purpose

Find, verify, restore, archive, and recoverably trash local Codex tasks through Codex's native App Server protocol. Success means a user can identify the exact task, preserve its transcript, perform one reversible action, and confirm the result without uploading data or editing Codex databases.

## Brand Personality

Forensic, calm, accountable. The tool speaks plainly, exposes evidence, and never turns a destructive operation into a casual click.

## Anti-references

No generic SaaS dashboard, AI assistant theatre, decorative analytics, purple gradients, oversized empty cards, or vague success messages. Do not imitate Codex's private UI or imply OpenAI endorsement.

## Design Principles

1. Evidence before action.
2. Native protocol before filesystem fallback.
3. Recovery is reversible and attributable.
4. Dense records should remain scannable by project.
5. Operational guidance belongs inside the tool.

## Accessibility & Inclusion

Meet WCAG 2.2 AA contrast, provide full keyboard navigation and visible focus, never encode state by color alone, respect reduced motion, and keep recovery instructions understandable without technical expertise.
