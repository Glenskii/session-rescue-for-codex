# Session Rescue for Codex: Notice

Created and maintained by **Glen E. Grant**. Glen E. Grant is Contributor #1 and the original creator of this Codex edition.

## Required restart after changes

After restoring or archiving a task, **fully close Codex Desktop and start it again**. Closing only the current task tab is not sufficient. Codex may keep stale sidebar state even after the transcript has been restored and verified successfully.
